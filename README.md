## 简介 / Introduction

本项目收录了 XDOJ 平台的部分题解。

- **双语支持：** 为每道题目提供了 C 和 C++ 两个版本的解答，并充分利用语言的原生特性。
- **工程规范：** 坚持统一的代码风格，并内置了自动化脚本与一站式工具集，确保代码的高维护性。
