# 拼数字

编号：108<br/>
难度：★★★☆☆<br/>
时间限制：1 秒<br/>

## 问题描述

对于给定的字符序列（长度小于 $100$），从左至右将所有数字字符取出拼接成一个无符号整数（输入保证拼接出的整数小于 $2^{31}$）。然后计算并输出该整数的最大因子（如果是素数或 $0$，则其最大因子为自身）。

## 输入说明

在一行输入一个字符序列（序列长度小于 $100$）。

## 输出说明

对输入的字符序列，求出所得整数的最大因子；若字符序列中没有数字或找出的整数为 $0$，则输出 $0$。

## 输入样例 1

```plain
sdf0ejg3.f?9f
```

## 输出样例 1

```plain
13
```

## 输入样例 2

```plain
?4afd0s&2d79*(g
```

## 输出样例 2

```plain
857
```

## 输入样例 3

```plain
abcde
```

## 输出样例 3

```plain
0
```
