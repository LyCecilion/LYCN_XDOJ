# 表达式求值

编号：77<br/>
难度：★★★☆☆

## 问题描述

这里的表达式由两个非负整数 `x`、`y` 和一个运算符 `op` 构成，需要计算表达式的值。

输入的表达式中，运算符可以在 `x`、`y` 之间、之前或之后，即 `x op y`，`op x y` 或者 `x y op`。

例如，`25 + 3` 表示 $25$ 加 $3$，`5 30 *` 表示 $5$ 乘以 $30$，`/ 600 15` 表示 $600$ 除以 $15$。

## 输入说明

输入为一个表达式，包含两个非负整数 `x`、`y` 和一个运算符 `op`，`x`、`y` 和 `op` 之间以空格分隔，`op` 可以在 `x`、`y` 之间、之前或之后。

`x` 和 `y` 的值均不大于 $10,000,000$，`op` 可以是 `+`、`-`、`*`、`/`、`%` 中的任意一种，分别表示加法、减法、乘法、除法和求余。除法按整数除法求值，输入数据保证除法和求余运算的除数 `y` 不为 $0$。

## 输出说明

在一行上输出一个整数，表示所输入表达式的值。

## 样例数据

### 输入样例 1

```text
5 20 *
```

### 输出样例 1

```text
100
```

### 输入样例 2

```text
4 + 8
```

### 输出样例 2

```text
12
```

### 输入样例 3

```text
/ 8 4
```

### 输出样例 3

```text
2
```
