# 找奇偶数的数量

编号：759<br/>
难度：★☆☆☆☆<br/>
时间限制：1 秒<br/>
内存限制：256 KB

## 问题描述

给定一个整数数组，请找出该数组中的奇数和偶数的个数。注意，本题要找的奇数和偶数是指大于等于 $0$ 的正整数。

## 输入格式

输入的第一行包含一个整数 $n$，表示数列中整数的个数。

第二行包含 $n$ 个整数 $a_1, a_2, \cdots, a_n$，表示给定的数列，相邻的整数之间用一个空格分隔。

## 输出格式

输出两个整数，表示奇数和偶数的个数。

## 样例输入

```plain
5
1 2 3 4 5
```

## 样例输出

```plain
3 2
```

## 样例说明

$1, 3, 5$ 是奇数，共 $3$ 个。$2, 4$ 是偶数，共 $2$ 个。所以输出是 `3 2`。

## 评测用例规模与约定

$1 \leq n \leq 1000, \quad -999 \leq a_i \leq 999$。
